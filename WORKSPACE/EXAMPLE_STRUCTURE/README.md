# Claude Code Complete File Scaffolding

This folder contains a complete starter scaffold for a Claude Code operating environment split into three layers:

1. `enterprise-system/` — administrator-managed policies and modular drop-in restrictions.
2. `user-home/.claude/` — global user preferences, personal skills, agents, commands, hooks, workflows, and per-project memory.
3. `project-root/your-project/` — repository-local Claude instructions, settings, skills, rules, agents, hooks, workflows, and isolated worktree space.

All secrets, tokens, and machine-specific values are placeholders. Review every file before use.
