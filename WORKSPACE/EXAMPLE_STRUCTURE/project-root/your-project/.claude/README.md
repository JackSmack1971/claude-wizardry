# Project Claude Directory

This directory contains repository-local Claude Code scaffolding.

- `settings.json` — shared project settings.
- `settings.local.json` — personal overrides; gitignored.
- `rules/` — path-scoped modular rules.
- `skills/` — reusable project workflows with scripts and references.
- `agents/` — focused subagent definitions.
- `commands/` — custom slash commands.
- `output-styles/` — communication styles.
- `hooks/` — executable gates and notifications.
- `workflows/` — deterministic JavaScript orchestration.
- `worktrees/` — isolated branch workspaces for parallel work.
