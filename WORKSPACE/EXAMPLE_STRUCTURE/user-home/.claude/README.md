# Global User Claude Directory

This folder contains user-wide Claude Code preferences and reusable automation:

- `CLAUDE.md` — global instruction layer.
- `settings.json` — global tool defaults and blocks.
- `.claude.json` — placeholder global state file; never commit real tokens.
- `rules/` — unconditional global habits.
- `skills/` — reusable personal workflows.
- `agents/` — reusable subagent profiles.
- `commands/` — slash command macros.
- `output-styles/` — communication styles.
- `hooks/` — executable lifecycle gates.
- `workflows/` — deterministic orchestration scripts.
- `projects/` — per-project auto-memory.
